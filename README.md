# Servlet_for_Restaurant_App

Educational Web App for Restaurant.

Main functionality:

Opens Web App and shows Menu from XML files (parsing with SAX, StAX, DOM).

Used Technologies:

  1) Servlet and JSP
  2) SAX, StAX, DOM parsers
  3) Layered architecture
  4) MVC
  5) Log4j2
